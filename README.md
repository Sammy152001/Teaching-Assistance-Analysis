# Teaching Assistance Analysis

## Description
This project analyzes teaching assistant data and applies Machine Learning techniques to evaluate and predict performance.

## Tools & Technologies
- Python (pandas, numpy, matplotlib)
- Machine Learning (scikit-learn)

## Dataset
Dataset is available in the `data/` folder.

## Objective
To analyze and predict teaching assistant performance using Machine Learning.

## Steps Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature selection  
- Model building using classification algorithm  
- Model evaluation  

## Key Insights
- Certain attributes strongly influence performance  
- Patterns observed in evaluation data  

## Machine Learning
- Model Used: Classification Model (e.g., Decision Tree / Logistic Regression)  
- Used to predict teaching assistant performance  

## Final Result
- Built a classification model  
- Achieved prediction based on dataset features  

## Conclusion
This project shows how Machine Learning can be applied in educational data analysis.
